# 📚 Library Management System

A Django-based digital library management system designed to reduce the time taken by students to search for books in a library.

## 🎯 Project Objective

The main objective of this project is to provide a digital platform for managing library resources efficiently. Students can quickly search for books, while librarians can manage book records through a dedicated interface.

## ✨ Features

### Student Module
- Student Login
- Search Books
- View Available Books
- Logout

### Librarian Module
- Librarian Login
- Add New Books
- Update Book Details
- Manage Library Records
- Logout

## 🛠 Technologies Used

- Python
- Django
- SQLite
- HTML
- CSS

## 🚀 How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/SuprajaPulavarthi07/library-management.git
```

2. Navigate to the project folder

```bash
cd library-management
```

3. Run the server

```bash
python manage.py runserver
```

4. Open your browser and visit

```text
http://127.0.0.1:8000/
```
